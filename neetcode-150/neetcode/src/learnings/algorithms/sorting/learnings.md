- Selection Sort

- Merge Sort
- Quick Sort
- Heap Sort
- Counting Sort
- Radix Sort
- Bucket Sort

## Bubble Sort

Just like a bubble rises up, the maximum value rises to the end of the array.
- Best: O(n)
- Worst: O(n^2)
- Average: O(n^2)
